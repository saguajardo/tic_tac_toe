### Install

Download or clone this repo with ssh:

```bash
git clone git@github.com:saguajardo/tic_tac_toe.git
```

Execute setup file to run docker

```bash
./setup
```

Grant write permissions to the following directories:

  * /storage
  * /bootstrap
